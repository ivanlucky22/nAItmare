# Workflow: Bug Fix

> **GOAL**: Resolve a reported issue.

## Steps
1.  **Reproduce**: Create a test case that fails (utilizing `test.md`).
2.  **Branch**: Create `fix/` branch using `git.md`.
3.  **Fix**: Modify code to resolve the issue.
4.  **Verify**: Run the new test case to confirm passing.
5.  **Commit**: Save work using `git.md`.
