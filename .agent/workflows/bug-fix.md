# Workflow: Bug Fix

> Resolve a reported issue.

## Steps
1. Reproduce: Create failing test case (see [test.md](../skills/test.md)).
2. Branch: Create `fix/` branch (see [git.md](../skills/git.md)).
3. Fix: Modify code to resolve issue.
4. Verify: Run test to confirm fix.
5. Commit: Save work with conventional commit.
