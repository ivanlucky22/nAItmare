# Testing Operations

## Commands
- Unit tests: `npm run test:unit` (Jest)
- E2E tests: `npm run test:e2e` (Playwright)

## Requirements
- Maintain >80% coverage on logic
- Do not proceed if tests fail

## On Failure
STOP. Report the failing test. Do not continue until resolved.
